Choate Labs is an independent software lab. Most of the catalog is tools for people who work with networks: a desktop terminal that takes Aruba gear seriously, MAC address lookup that works in a basement with no signal, rack documentation built around LiDAR.

The rest escaped from the lab notebook: a trading-bot platform, a self-hosted dashboard for the home lab, and an arcade game. Everything ships from a one-person operation.
