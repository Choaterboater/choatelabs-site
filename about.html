Choate Labs is the independent software practice of Stephen Choate. Native iOS. No venture capital. No roadmap theater. No AI slop. Just software that ships.

The Foundry

Most apps are built to capture attention. Choate Labs builds them to map connection — between people, between assets, between ideas. Every product on the bench is an experiment in network topology.

MAC Vendor Lookup started because every IT pro alive has typed an OUI into a search box one too many times. Threadback started because genealogy software still looks like tax software. The trade tools started because the market is the most brutal social network alive, and nobody has built a decent graph for it.

If it doesn't fit in your pocket, it doesn't ship. If it needs a tutorial, it's broken. If it sells your data, we don't build it.

Operating Principles

01

Network-first design

Everything is a graph. Family trees, server racks, MAC tables, price action. The interface should reveal the topology, not bury it.

02

Privacy by default

No data brokers. No ad networks. No "anonymized" enrichment. Your network is yours; we are just the lens.

03

Native only

iOS is the substrate. Built for Metal, for SwiftData, for the constraints of actual silicon. Not Electron. Not React Native. Swift and sweat.

04

Ship, then iterate

Version 1.0 is a hypothesis. Version 2.0 is the experiment confirmed. We do not wait for perfect; we wait for coherent.
