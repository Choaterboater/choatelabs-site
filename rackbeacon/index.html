Document any rack, any closet, any time.

RackBeacon turns your iPhone into the field tool you wish you'd had on day one — camera-scan serials and MACs, build U-by-U rack layouts, capture floor plans with LiDAR. All offline.

Field Capabilities

▤

Rack visualization

Build racks from 4U to 52U with port-accurate faceplates — copper, SFP, SFP+, QSFP. Drag a device to its U, see exactly what plugs where.

⊟

Pre-loaded device library

Aruba CX 6000–10000, Cisco Catalyst 9k / Nexus / ISR / ASR / Firepower / UCS, Juniper EX/QFX/SRX/MX, Fortinet, Palo Alto PA, HPE ProLiant, APC Smart-UPS — already modeled.

▢

Camera scanner

On-device Vision OCR pulls serial numbers, model strings, MACs, BLE MACs, and IMEIs straight off device labels. Code 128, Code 39, Data Matrix, QR.

⌖

Floor plan capture

Walk a comms room with a LiDAR-equipped iPhone Pro or iPad Pro and Apple RoomPlan returns a clean 3D plan. Multi-room scans stitch into one building.

⌬

Annotations & cable runs

Drop racks, devices, notes, and cable runs onto the floor plan. The closet finally looks like a network, not a memory.

⇄

CSV / JSON / SVG export

Per-rack CSV (NetBox-friendly), full-site JSON, and floor plans as SVG. CSV import means round-trips with the rest of your tooling.

◎

100% offline

No accounts. No analytics. No third-party SDKs. No network code in the binary. Your rack, your data, your device.

⊕

Universal app

iPhone for the field, iPad for the close-up. Same project file, same camera scanner, same export. Built native for both form factors.

No data leaves the lab.

Built private by default.

RackBeacon is fully offline. There is no network code in the binary. Your rack layouts, floor plans, photos of device labels, notes, and serials stay on your device.

No accounts No tracking No analytics No third-party SDKs No advertising No telemetry
