choatelabs.app / greencli

Project 01 · macOS & Windows

Every vendor. One terminal.

GreenCli is a macOS SSH/Telnet/Serial terminal built for network engineers who live in multi-vendor environments. Aruba, Juniper, Mist — syntax-aware, AI-assisted, and vault-encrypted.

macOS & Windows · Tauri Desktop · v1.0.0

⬇ macOS (Apple Silicon) macOS (Intel) Windows (x64)

Unsigned beta builds — macOS: right-click the app → Open on first launch · Windows: "More info" → "Run anyway"

Vendor support

Aruba CX Switches Aruba AOS-S Switches Aruba APs Aruba Controllers Juniper JunOS Mist Generic SSH / Telnet / Serial

What's under the hood

⌗

Multi-vendor syntax highlighting

Real-time ANSI-injected highlighting for Aruba CX, AOS-S, APs, controllers, and JunOS. Auto-detects device type from the prompt.

⚡

AI assistant

Send commands from the AI panel, get config explanations, and run troubleshooting flows. Supports Anthropic, OpenAI-compatible endpoints, and local Ollama models.

🔐

Encrypted credential vault

Passwords and keys stored with AES-256-GCM + Argon2. Nothing leaves the machine unencrypted.

⛓

MCP integration

Connects to MCP servers like centralmcp to give the AI assistant live Aruba Central access — device state, NAC policies, monitoring alerts, and more.

🌐

REST API explorer

Browse and fire API calls against Aruba CX, AOS-S, and Juniper Apstra directly from the app — no Postman, no browser tab switching.

⇄

SSH tunnels

Local and SOCKS5 dynamic port forwarding built in. Manage tunnels from the title bar without touching the CLI.

📁

SFTP file ops

Browse remote filesystems, upload with native file-drop, rename, and delete — without leaving the terminal window.

🗂

Session manager

Folders, tags, fuzzy search, and jump hosts. Import directly from your existing SSH config file.

✎

Config editor

Monaco-powered editor with per-vendor config templates. Write, validate, and push configs without leaving the app.

No cloud required.

Your network stays your network.

GreenCli runs entirely on your Mac. Credentials live in a local vault. AI calls go directly from your machine to your chosen provider — no proxy, no telemetry.
