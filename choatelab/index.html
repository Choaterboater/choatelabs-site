Project 07 · Self-hosted · Open source

One pane of glass for the home lab.

ChoateLab is a self-hosted dashboard for the gear, services, and signals that make a home lab feel like a network. Docker, Plex, Sonarr, Pi-hole, Home Assistant, Aruba — drag the cards you care about onto your home screen, ignore the rest.

Self-hosted · Docker / Swarm / Native · Calibrating

View on GitHub See Features

What's on the bench

▦

Builder layout

Drag-and-drop grid where you choose which services show up on the home page. Nothing appears by default — your dashboard, your call.

⏣

22+ integrations

Media servers, *arr stack, Docker, Pi-hole, qBittorrent, Home Assistant, Grafana, Prometheus, TrueNAS, OPNsense, Aruba Central, Juniper Mist, and more.

◫

Service detail drawers

Click any card to drill into containers, libraries, queues, alerts, and quick actions — without leaving the dashboard.

⊕

"Add Stuff" quick search

Floating action button to search and add movies (Radarr), TV (Sonarr), or torrents (qBittorrent) in a couple of keystrokes.

⌁

Live system metrics

CPU, RAM, disk, and network I/O with sparkline charts. Real-time updates over WebSockets.

◷

Resilient by default

Request deduplication, exponential backoff on errors, 30 s health-check cache, process-level crash protection. No thundering herd, no silent dead backend.

⧉

Single deploy

One Docker image serves the API and the SPA on one port. Compose for one host, Swarm for many. SQLite for state — nothing else to run.

⌬

Open source

MIT-licensed. React + TypeScript on the front, Node + Express on the back. PRs welcome.

Built honest, built fast.

The stack is the point.

No framework soup, no telemetry, no SaaS dependency. React 19, Vite 7, Tailwind, shadcn/ui, Framer Motion, Recharts on the client. Node 20, Express 5, better-sqlite3, ws on the server. Everything fits in one repo and one container.
